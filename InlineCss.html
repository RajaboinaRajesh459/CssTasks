Inline CSS

An inline style may be used to apply a unique style for a single element.

To use inline styles,add the style attribute to the relevent element.the style attribute can contain any CSS property.

Css file saved as .css like:"style.css"

EX:<h1 style="color:blue;">This is a heading<h1>


Internal CSS

An internal style sheet may be used if one single HTML page has a unique style.

The internal style is defined inside the <style>element,inside the head section.

Ex:<style>
body{
background-color :blue;
}


External Css

With an external style sheet,you can change the look of the entire website by changing just one file!.

Each Html page must include a reference to the external style sheet file inside the <link> element,inside the head section

Ex:<link rel="stylesheet" href="style.css">
